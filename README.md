# SIT 起身动动 🪑➡️🏃

> 一个专为久坐办公族设计的网页工具，定时提醒你起身活动，并指导你完成 15 个科学工位微运动。

## 🌐 在线使用

👉 **[打开网站](https://yxx6666.github.io/sit-reminder/)**

（无需安装，浏览器打开即用）

## ✨ 核心功能

- ⏱️ **智能计时器** — 30/45/60 分钟预设，自定义倒计时
- 🔔 **多重提醒** — 提示音 + 桌面通知 + 标签闪烁 + 兜底弹窗
- 🏃 **15 个微运动** — 肩颈/手臂/全身，每个配动作说明
- 🧘 **478 呼吸法** — 吸4-屏7-呼8，3 轮引导动画
- 📊 **数据统计** — 今日起身次数、连续天数、本周趋势柱状图
- 🏅 **成就系统** — 连续 3/7/21 天解锁成就 + 彩纸特效
- 📤 **健康报告** — 一键生成周报，支持打印/存 PDF
- 🌙 **暗色模式** — 手动切换或跟随系统
- 📱 **响应式** — 手机/平板/桌面全适配

## 🛠 技术实现

- 纯 HTML/CSS/JS 单文件应用
- SVG 环形进度条 + 呼吸光效动画
- Web Audio API 提示音
- Notification API 桌面通知
- Canvas 分享卡片生成
- localStorage 数据持久化
- GitHub Pages 部署
