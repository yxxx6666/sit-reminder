## ❤️ 为什么要使用这个工具？

### 你可能正在经历这些：

- 🪑 早上坐下，回过神来已经中午了，腰酸背痛
- 📱 市面上的提醒 App 只会弹个通知，关掉继续坐着
- 🤷 知道久坐不好，但不知道该起来做什么动作
- 📉 长期下来，颈椎反弓、腰椎突出、手腕酸痛找上门

### 这个工具能帮你：

- ✅ **不只是提醒，而是指导**：休息弹窗里内置 15 个科学工位微运动，每个都适合在办公室做
- ✅ **给你坚持的动力**：连续天数追踪 + 成就系统，让起身变成一种养成游戏
- ✅ **让你看到变化**：本周趋势柱状图、健康周报，数据可视化你的健康改善
- ✅ **温柔的督促**：幽默文案 + 呼吸光效，不是暴力打断，而是温暖提醒

### 科学依据：

每坐 45-60 分钟起身活动 2-3 分钟，可显著降低心血管疾病风险（美国糖尿病学会，2023）；每 30 分钟起身走动 2 分钟，可有效改善血糖控制和血压（澳大利亚贝克研究所）。

## ✨ 核心功能

### ⏱️ 智能计时
- 30/45/60 分钟预设 + 自定义秒数
- SVG 环形进度条 + 呼吸光效动画（颜色从绿到红，直观感知剩余时间）
- 页面标题实时倒计时（切换标签也能看到）

### 🔔 多重提醒
- Web Audio API 六音提示音（C-E-G 和弦，轻柔不刺耳）
- Notification API 桌面通知（离开页面也能收到）
- 标签闪烁 + 卡片高亮动画
- 页面不可见时兜底弹窗提醒

### 🏃 15 个工位微运动
- **上半身·肩颈手臂**：靠墙天使、坐姿脊椎扭转、颈部环绕、肩部环绕、手腕旋转、手臂交叉伸展、扩胸运动
- **全身·站立拉伸**：站立猫牛式、踮脚站立、站立侧弯、椅子俯卧撑、深蹲
- **坐姿·腿部脚踝**：坐姿抬腿、坐姿前屈、脚踝旋转
- 独立运动说明页，分类展示，点开即学

### 🧘 478 呼吸法
- 吸气 4 秒 → 屏息 7 秒 → 呼气 8 秒，3 轮循环
- CSS 动画圆圈引导节奏，帮助快速放松

### 📊 数据统计
- 今日提醒次数 / 已休息次数 / 已完成活动数 / 连续天数
- 本周趋势纯 CSS 柱状图（今天高亮）
- 今日跳过次数（含跳过原因记录）
- 今日活动记录（最近 5 条）

### 🏅 成就系统
- 🎖️ 连续 3 天：初露锋芒
- 🏆 连续 7 天：脊柱守护者
- 👑 连续 21 天：久坐克星
- 解锁时彩纸飘落动画特效

### 🦴 脊椎状态指示器（桌面端）
- 右下角 SVG 脊椎示意图
- 累计久坐超 2 小时变黄，超 3 小时变红
- 休息后恢复绿色
- 气泡文字实时反馈状态

### 📤 健康报告
- 一键生成本周健康报告（A4 比例，支持打印 / 存为 PDF）
- 自动健康小结（5 个等级评语）

### 📤 分享卡片
- Canvas 生成今日打卡卡片
- 包含起身次数、连续天数、健康金句、底部署名

### 🎨 视觉与交互
- 莫兰迪暖色系配色（柔和护眼）
- 暗色模式（手动 / 跟随系统）
- 微纹理背景 + 柔和阴影
- 响应式布局（380px 以上全适配）
- 非阻塞 Toast 轻提示

### ⚙️ 自定义设置
- 提示音开关 / 桌面通知开关
- 护眼 20-20-20 提醒（每 20 分钟看远处）
- 暗色模式切换
- 自动开始计时
- 重置今日统计
